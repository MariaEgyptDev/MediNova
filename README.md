# Smart Healthcare Website

## Project Overview

This project is a simple healthcare website designed to provide users with basic medical information and an easy way to explore healthcare services. The goal of the website is to make health resources more accessible through a clean and user-friendly design.

## Features

* Simple and responsive design
* Easy navigation between pages
* Clear presentation of healthcare information
* Organized layout for better user experience

## Technologies Used

* HTML
* CSS

## How to Use

1. Download or clone the project files.
2. Open the **index.html** file in your web browser.
3. Navigate through the website pages.

## Project Purpose

This project was created as part of a learning process to practice web development and website design.
